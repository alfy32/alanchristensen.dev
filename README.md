# alanchristensen.dev

Personal homepage — name, tagline, and links to GitHub, LinkedIn, and [developer tools](https://tools.alanchristensen.dev).

## Stack

Plain HTML, CSS, and vanilla JavaScript. No framework, no build step, no dependencies.

## Running locally

Open `index.html` directly in a browser, or serve the root with any static file server:

```
npx serve .
```

## Hosting

GitHub Pages at [alanchristensen.dev](https://alanchristensen.dev). Push to `main` and it deploys automatically.

## Developer Tools

The tools section is a separate site and repo at [tools.alanchristensen.dev](https://tools.alanchristensen.dev).
